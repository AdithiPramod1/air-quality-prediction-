 AirAware - Air Quality Prediction System

 Description

AirAware is a machine learning project that predicts air quality based on environmental data.
It helps users understand pollution levels and take necessary precautions.


Objective

* Analyze air quality data
* Predict Air Quality Index (AQI)
* Provide an easy interface for users

 Modules
 
 1. Data Collection

* Dataset collected from Kaggle

 2. Data Processing

* Cleaned and preprocessed data
* Handled missing values

3. Model Training

* Used XGBoost algorithm
* Trained model for AQI prediction

4. Frontend

* Built using Streamlit
* User inputs data → system predicts air quality



Features

* Predicts AQI instantly
* Simple and user-friendly interface
* Fast and efficient model


 Technologies Used

* Python
* Pandas
* Scikit-learn
* XGBoost
* Streamlit


How to Run

1. Install required libraries:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py


 Output

* Displays predicted AQI value
